# HW2_TP_thereal
Проект ДЗ по ТП на тему ООП, тестирование работа с Git. Задача - разработка системы для управления рецептами.
В проекте реализованы классы для работы с рецептами: ингредиенты, сами рецепты, лист покупок и рецепт для диеты. Код демонстрирует основные принципы ООП: создание классов, атрибуты объектов, методы, свойства, проверки данных и сравнение объектов.

## Установка
Склонируйте репозиторий:
```bash
git clone https://github.com/therealars/HW2_TP_thereal.git
cd HW2_TP_thereal
```

## Использование
Проект состоит из отдельных Python-модулей с классами:
- `Ingredient.py` — класс ингредиента;
- `Recipe.py` — класс рецепта;
- `DietaryRecipe.py` — класс диетического рецепта;
- `ShoppingList.py` — класс списка покупок.
Основная проверка работы проекта выполняется через тесты:
```bash
pytest
```
